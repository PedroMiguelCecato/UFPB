# UFPB
---

Este projeto apresenta uma coletânea de atividades e conteúdos de programação desenvolvidos durante minha trajetória na UFPB (Universidade Federal da Paraíba). Ele funciona como um portfólio do meu trabalho acadêmico, demonstrando minhas habilidades e compreensão de diversos conceitos de programação. Embora tecnologias e funcionalidades específicas não estejam listadas, este repositório representa meu compromisso com o aprendizado e a aplicação do conhecimento em programação.
